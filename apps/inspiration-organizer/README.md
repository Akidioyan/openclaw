# 灵感集工具自动整理和索引系统

## 项目概述

一个智能化的灵感集管理系统，能够自动识别、分类、索引和推荐各种类型的内容（文本、图片、音频、视频）。

## 功能特性

- **智能分类**：AI自动识别内容类型和主题
- **自动索引**：建立搜索索引和关键词提取
- **智能推荐**：基于内容相似度推荐相关资讯
- **自动整理**：定期清理过期内容，整理重复信息

## 支持的内容类型

- 文本内容（文章、笔记、链接）
- 图片内容（设计作品、照片、截图）
- 音频内容（音乐、语音记录、播客）
- 视频内容（教程、演示、创意视频）

## 技术栈

- Python 3.8+
- NLTK - 自然语言处理
- Whoosh - 全文搜索引擎
- Pandas - 数据分析
- OpenAI GPT - AI分类
- Pillow - 图像处理
- librosa - 音频处理
- OpenCV - 视频处理
