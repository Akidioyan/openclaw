## 🎯 其他身份验证方案

### 方案一：使用 Bearer Token 身份验证（推荐）

**在代理会话之外执行**：

```bash
# 创建或编辑 ~/.xurl 文件
cat > ~/.xurl << EOF
---
version: 1
apps:
  openclaw:
    name: openclaw
    client_id: dummy
    client_secret: dummy
    tokens:
      - type: bearer
        token: YOUR_BEARER_TOKEN_HERE
EOF
```

**注意**：请替换 `YOUR_BEARER_TOKEN_HERE` 为您的实际 Bearer Token。

### 方案二：使用 OAuth 1.0a 身份验证

**在代理会话之外执行**：

```bash
# 删除当前应用程序
xurl auth apps remove openclaw

# 重新注册应用程序（OAuth1）
xurl auth apps add openclaw --consumer-key "YOUR_CONSUMER_KEY" --consumer-secret "YOUR_CONSUMER_SECRET"

# 进行 OAuth1 身份验证
xurl auth oauth1
```

### 方案三：使用 Application Only 身份验证

**在代理会话之外执行**：

```bash
# 使用 Application Only 身份验证
xurl auth apps add openclaw --client-id "YOUR_CLIENT_ID" --client-secret "YOUR_CLIENT_SECRET"

# 进行 Application Only 身份验证
xurl auth oauth2 --app-only
```
